# web-boilerplate

### Included:

##### JavaScript:
- Webpack
- Babel

##### CSS:
- SASS
- Autoprefixer

##### Utilities:
- BrowserSync
- Minifying

### Usage/Commands:
#### Start the task: 
- yarn start / npm run start
#### Build process:
- yarn build / npm run build
#### Minifying process:
- yarn minify / npm run minify
